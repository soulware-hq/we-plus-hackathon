## Description
Ben jij een probleemoplosser die energie haalt uit het bouwen van robuuste en schaalbare Java-applicaties? Heb je een passie voor het schrijven van propere, efficiënte code en werk je graag mee aan projecten die echt impact maken? Als je enthousiast wordt van principes zoals DDD, CQRS en Event Sourcing, dan is deze rol jou op het lijf geschreven.

Als Java Software Developer bij we+ werk je aan innovatieve oplossingen voor een uiteenlopende groep klanten. Je gebruikt een brede waaier aan tools en technologieën, en krijgt de ruimte om te innoveren in een dynamische en ondersteunende werkomgeving.

## Profile (what we look for)
+ Ervaring: Je hebt minstens 5 jaar ervaring als Java Software Developer – of je overtuigt ons waarom jij de juiste match bent, ook met minder ervaring.

+ Technische kennis: Je bent vertrouwd met beveiliging, webservices, Spring Boot, JavaScript, Hibernate, REST/SOAP, JSON/XML en SQL. Kennis van NoSQL is mooi meegenomen.

+ Mentaliteit: Je bent proactief, kritisch op een positieve manier en voelt je goed in een teamgerichte omgeving. Je communiceert vlot in Nederlands en Engels (Frans is een pluspunt).

+ Agile mindset: Agile werken zit in je DNA en je zoekt voortdurend naar manieren om je werkwijze te verbeteren.

## Offer (what we offer)
+ Dynamische cultuur: Een no-nonsense, ondersteunende bedrijfscultuur waarin groei op lange termijn en menselijke relaties centraal staan.

+ Aantrekkelijk loonpakket: Een competitief salaris aangevuld met extra’s zoals een bedrijfswagen, tankkaart, verzekering, maaltijdcheques en de beste tools om je werk goed te doen.

+ Professionele groei: Op maat gemaakte opleidingen, continue bijscholingskansen en uitdagende projecten met échte impact.

+ Flexibiliteit: Je werkt remote, op kantoor (Hasselt, Mechelen of Brussel), of bij de klant – afhankelijk van het project.

+ Teamspirit: Van jaarlijkse teambuildings tot onze legendarische Sinterklaasfeesten – bij we+ combineren we hard werken met plezier en verbinding.
