# VAC-006 — Junior/Medior DPO (NIS2, Vlaamse Overheid)

**Source:** Privatum partner intake (forwarded by Ellen Mollen, 2026-04-27)
**Sourcing partner:** Atmoz Consult (Sabine De Beule)
**End client:** Vlaamse Overheid (NIS2 traject)
**Start:** asap
**Duration:** ~25 dagen tot einde jaar (kans op verlenging)
**Language:** Nederlands C2

## Description
Wij zijn voor een **NIS2-traject met oplevering einde jaar** dringend op zoek naar een Junior/Medior DPO die ter ondersteuning van een senior profiel o.a. de huidige VO-policies tov de voorgestelde templates vergelijkt en de "low hanging fruit" kan plukken.

## Required skills

### Must have
- Aantoonbaar certificaat **ISO 27001 implementer / auditor** of **NIS2 implementer**.
- Aantoonbare ervaring met ISO 27001 — Jr (1j) / Mr (3j).
- Aantoonbare ervaring van security beheer technieken/raamwerken (ISO27k, CyFun, NIST, CIS Critical Security Controls) — Jr (1j) / Mr (3j).
- Aantoonbare ervaring als Security Consultant binnen één van: data, infrastructuur, applicaties — Jr (1j) / Mr (3j).
- Aantoonbare kennis in analyseren, optimaliseren en documenteren van security processen en governance.
- Aantoonbare kennis van information security.
- Taalvereiste: Nederlandstalig CEFR C2.

### Should have
- Ervaring met Agile/Scrum.
- Ervaring met NIS2.
- Ervaring binnen de Vlaamse Overheid.
- Ervaring met security beheer technieken/raamwerken (ISO27000, COBIT for Security, NIST, OWASP, CIS Critical Security Controls).
- Certificaten zoals CISM, CISSP, CEH afhankelijk van expertisedomein.
